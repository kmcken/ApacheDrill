NI cRIO 9047
NI 9237
NI 9422
NI 9203

Software Dependencies:
LabVIEW 2019 SP1 
LabVIEW Real-Time Module 2019
LabVIEW FPGA Module 2019
NI-DAQmx 19.0
NI CompactRIO 19.5
